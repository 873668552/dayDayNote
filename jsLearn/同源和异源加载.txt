 同源加载没啥说了。主要说的是异源。
异源：就是不同网站之间的数据交互，具体的说就是a客户端请求b服务器的数据。
1.
script 的特性。前端引用资源的一个标签，不限制资源的来历。所以把需要的数据
URL写在script的src属性中。就可以达到加载异源数据的目的。
2，用同源的服务器（后端）加载别的服务器资源，然后传递到同源前端。
用后端的方法。如php：$result = file_get_contents("http://x.x.x65/1.php");
3,CORS跨域资源共享
<?php
// 下面这句话指允许来至任何人的访问
header('Access-Control-Allow-Origin: *');   // 访问 控制 允许 起源
// 允许特定的网址来访问
header(‘Access-Control-Allow-Origin:http://127.0.0.1:8020’);	// 多个网址时，一行一行写。
