用for创建组件的时候，控件事件的绑定用this指向控件，不要用循环中的变量找控件。
原因：如 for(var i = 0; i < 3; i++)
	{
		btn[i].onclick = function(){
			btn[i].value = i;
		}
	}
	第一个btn[i] 和 函数里面的btn[i] 指向的不是同一个控件.第二个指向的是btn[2].
	函数的执行是在所有的js块中的代码都执行完之后，才去执行的。
